# @fka/prettier-config

> The prettier configuration of the FKA team

## Usage

**install:**

```bash
npm install @fka/prettier-config -D
```

```bash
pnpm install @fka/prettier-config -D
```

```bash
yarn add @fka/prettier-config -D
```

**Edit package.json:**

```json
{
  // ...
  "prettier": "@fka/prettier-config"
}
```
